# Mon Panier

## Description

Mon Panier est une application web développée en PHP et MySQL permettant de gérer des listes de courses et un stock de produits.

Ce projet a été réalisé dans le cadre de ma formation de Développeur Web.

## Fonctionnalités

- Inscription
- Connexion
- Création de listes de courses
- Gestion des produits
- Gestion du stock
- Gestion du profil utilisateur

## Technologies utilisées

- HTML5
- CSS3
- Bootstrap 5
- PHP
- MySQL
- XAMPP
- Git
- GitHub
- Figma

## Installation

1. Cloner le dépôt GitHub.
2. Copier le projet dans le dossier `htdocs` de XAMPP.
3. Importer le fichier `mon_panier.sql` dans phpMyAdmin.
4. Démarrer Apache et MySQL.
5. Ouvrir le navigateur à l'adresse :

```
http://localhost/mon_panier
```
